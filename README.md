# LifePit Partner Dictionary

LifePit partner site built with Next.js and Tailwind CSS.

## Local development

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:3000`.

## Deploy with Vercel

1. Create a new GitHub repository.
2. Push this project to GitHub.
3. Import the repository into Vercel.
4. Deploy with the default Next.js settings.

## Recommended

- Use a normal local folder, not a Google Drive synced folder.
- Use Node.js 22 LTS on GitHub Actions / Vercel.
